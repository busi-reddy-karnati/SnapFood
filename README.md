# SnapFood

SnapFood is a food-focused application — snap a photo of a meal and get details about it.

> This repository is in early development. Setup and usage instructions will expand as the codebase grows.

## Getting Started

### Prerequisites

- Git

### Setup

```bash
git clone https://github.com/<your-username>/SnapFood.git
cd SnapFood
```
